# Routelle
A simple simulation of a slot machine 

## Usage
Clonez le repos git et compilez le fichier **routelle.c**.

## Commandes

> !h // affiche l'aide
> !c // donne 200 de crédit
> !p \<mise> // lance la roulette avec la mise donné. mise par défaut de 5
