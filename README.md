# Routelle
A simple simulation of a slot machine 

## Usage
Clonez le repos git et compilez le fichier **routelle.c**.

## Commandes

> !h // affiche l'aide <br>
> !c // donne 200 de crédit <br>
> !p \<mise> // lance la roulette avec la mise donné. mise par défaut de 5 <br>
